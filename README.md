# Paper & Clay

A warm, low-glare dark theme for VS Code — terracotta, sage, ochre, and slate on aged paper. Tuned for long coding sessions.

Originally bundled with the [Code Workbench](https://github.com/tesenwein/code-workbench-extension) extension, now a standalone theme.

## Install

### From VSIX

1. Build or download `paper-and-clay-theme-<version>.vsix`.
2. In VS Code: **Extensions** view → `⋯` menu → **Install from VSIX…** → select the file.
3. **Preferences: Color Theme** → **Paper & Clay**.

### Build a VSIX

```sh
npm run package
```
